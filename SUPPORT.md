# Support RS3-DataTool

RS3-DataTool is a free, unofficial, local-first RuneScape 3 desktop reference utility.

It provides read-only lookup tools for Grand Exchange items, Bestiary monsters, public player hiscores, and RuneScape Wiki pages.

## Ways to support

You can support the project by:

* Starring the repository
* Reporting bugs
* Suggesting useful features
* Sharing it with other RuneScape 3 players
* Testing release builds
* Donating through the project support link

## What support helps with

Support helps maintain and improve:

* Grand Exchange lookup tools
* Bestiary reference tools
* Public profile skill lookups
* Wiki Portal search and saved references
* Windows desktop packaging
* Cross-platform build research
* Local storage and persistence features
* Documentation, testing, and release maintenance

## Project boundaries

RS3-DataTool is a read-only reference companion.

It does not:

* Automate gameplay
* Provide macros
* Hook into the RuneScape client
* Modify game files
* Act as an overlay
* Ask for Jagex login credentials
* Collect account credentials

Only public or user-saved reference data is used.

RS3-DataTool is unofficial and is not affiliated with, endorsed by, or sponsored by Jagex.
